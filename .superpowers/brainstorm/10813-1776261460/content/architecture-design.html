06 — Error Handling
Invalid URL
Caught in parseSpotifyUrl() before any API call. Immediate error state, no network request.
Playlist not found / private
Spotify returns 404 or 403. Mapped to user-friendly message. "This playlist doesn't exist or is private."
API failure
Network error or 5xx. Generic fallback message with retry option.
